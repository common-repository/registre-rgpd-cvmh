== Installation ==
1. Unzip the plugin and upload the "cvmh-registre-rgpd" folder to your "/wp-content/plugins/" directory
2. Activate the plugin through the "Plugins" administration page in WordPress
3. Use the shortcode or the widget
4. Enjoy.
